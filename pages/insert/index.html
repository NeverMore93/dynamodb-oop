Insert Item

Insert is handled as putItem with an extra condition to make sure item does not already exist.

Insert does not replace existing items. Use .insert_or_replace() or .insert_or_update() instead.

WARNING: insert() will do an extra call (describeTable) to get the table schema and prevent item overwrite,
If an item with the same key exists, 'ConditionalCheckFailedException' error is returned

DynamoDB .table('users') .insert({ // carefull though as foo.bar domain actually exists :) domain: 'foo.bar', email: 'baz@foo.bar', password: 'qwert', boolean: true, number: 1, created_at: new Date().getTime(), updated_at: null, buffer: new Buffer("test"), array_empty: [], // inserted as datatype L array_strings: ['alfa','beta','gama'], // inserted as datatype SS string_set: DynamoDB.SS(['sss','bbb','ccc']), // inserted as datatype L array_numbers: [7,9,15], // inserted as datatype NS number_set: DynamoDB.NS([111,222,333]), array_mixed: [ null, "string", 5, true, false, { key: "value"}, ["nested","array"], new Buffer("test") ], nested_object: { name: "Foo", email: "baz@foo.bar", nested_attribute: { boolean_value: true, null_key: null, some_string: "tadaa", lucky_number: 12 } } }, function(err,data) { });
